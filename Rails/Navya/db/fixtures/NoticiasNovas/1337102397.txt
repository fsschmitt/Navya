Interessado em desenvolver aplicações para a nova versão do Windows? A Microsoft organiza no próximo dia 28 de maio no Porto o Windows 8 App Camp, um evento dirigido a developers que promete partilhar toda a informação necessária para criar aplicações para a versão do Windows apresentada pela Microsoft como a maior revolução desde o Windows 95
